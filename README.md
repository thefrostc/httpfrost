# PyShare

Secure and simple Python-based LAN file sharing HTTP server with upload, selective ZIP download, and multi-file support.

---

## Features

- Serve files and directories over your local network easily.
- Browse directories with file sizes and multi-select for download.
- Download individual files directly, or folders compressed as ZIP archives.
- Upload files through the web interface securely.
- Prevents directory traversal and common security pitfalls.
- Lightweight and zero dependencies outside the Python standard library.
- Runs with a single command and customizable port.

---
